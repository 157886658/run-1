## 准备工作:

### 护照准备: 

- 现在大部分省份护照不给发了(正当理由也不行). 要想办法搞出来.
- 花钱+走关系, 不管何种手段, 先把护照搞出来.  

### 资产准备: 

- 保底: 40-100w RMB
- 家境好, 跳过.
- 家境不好, 先工作赚钱, 存钱+理财(推荐数字货币).



### 签证申请: 

- 旅游签证: 部分中介, 有旅游签证入国+申请学校(转学签), 再毕业拿工签. 这也是一种途径(不可控).
- 留学签证: 选学校, 递材料, 通过, 办签证.
- 工作签证: 选企业, 入职, 工作一段时间, 办签证.
